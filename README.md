# chat.app

## Installation

Use the requirements.txt file to install all of the dependencies.

### macOS

```bash
pip3 install -r requirements.txt
```

### Windows

```bash
pip install -r requirements.txt
```

## Usage

### macOS

```bash
python3 wsgi.py
```

### Windows

```bash
python wsgi.py
```

Then you can go to https://localhost:5000 to run and test the app.
